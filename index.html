1. JavaScript คืออะไร

ความหมาย

JavaScript (JS) คือ ภาษาคอมพิวเตอร์ในรูปแบบสสคริปต์ (Scripting Language) ที่ถูกพัฒนาขึ้นมาเพื่อทำให้หน้าเว็บไซต์มีความเป็น Dynamic หรือมีการเคลื่อนไหว ตอบสนองกับผู้ใช้งานได้แบบเรียลไทม์ ต่างจากภาษา HTML และ CSS ที่ทำหน้าที่เพียงจัดโครงสร้างและหน้าตาที่หยุดนิ่ง (Static)

ประวัติความเป็นมา

จุดเริ่มต้น (1995): JavaScript ถูกสร้างขึ้นโดย Brendan Eich ในปี ค.ศ. 1995 ขณะที่เขาทำงานให้กับบริษัท Netscape Communications โดยในตอนแรกเริ่มใช้ชื่อว่า Mocha จากนั้นเปลี่ยนเป็น LiveScript และสุดท้ายตกลงใช้ชื่อ JavaScript เพื่อเกาะกระแสความนิยมของภาษา Java ในยุคนั้น

ความสำคัญของ JavaScript

  • เป็นหัวใจของเว็บยุคใหม่: เว็บไซต์ระดับโลกอย่าง Facebook, YouTube หรือ Google ล้วนขับเคลื่อนด้วย JavaScript เพื่อให้ผู้ใช้สามารถกดไลก์ คอมเมนต์ หรือโหลดข้อมูลใหม่ได้โดยไม่ต้องกด Refresh หน้าเว็บ
  • เรียนรู้ครั้งเดียว เขียนได้ทุกอย่าง (Universal Language): ด้วยการมาของ Node.js ทำให้เหล่านักพัฒนาสามารถใช้ JavaScript เขียนได้ทั้งฝั่งหน้าบ้าน (Frontend) และหลังบ้าน (Backend) รวมถึงทำแอปพลิเคชันบนมือถือ (React Native) ได้อีกด้วย
  • Ecosystem ที่แข็งแกร่ง: มี Library และ Framework ยอดนิยมมากมาย เช่น React, Vue, และ Angular ที่ช่วยให้การพัฒนาซอฟต์แวร์รวดเร็วและมีประสิทธิภาพ

2. การทำงานของ JavaScript

Client-side

การทำงานของ JavaScript ส่วนใหญ่จะเป็นแบบ Client-side หมายความว่าตัวโค้ดจะถูกดาวน์โหลดจากเซิร์ฟเวอร์มาพร้อมกับหน้าเว็บ และมาประมวลผลที่เครื่องของผู้ใช้งาน (Client) โดยอาศัยเว็บเบราว์เซอร์ ข้อดีคือช่วยลดภาระการทำงานของเซิร์ฟเวอร์ และทำให้การตอบสนองต่อผู้ใช้งานทำได้อย่างรวดเร็วฉับไว

การทำงานร่วมกับ HTML และ CSS

JavaScript จะเข้าไปควบคุม HTML และ CSS ผ่านสิ่งที่เรียกว่า DOM (Document Object Model) ทำให้เราสามารถสั่งให้ข้อความเปลี่ยนสี (เปลี่ยน CSS) หรือสั่งให้ปุ่มกดแล้วมีกล่องข้อความเด้งขึ้นมา (เปลี่ยน HTML) ได้

การประมวลผลผ่าน Web Browser

ภายในเว็บเบราว์เซอร์ (เช่น Google Chrome, Microsoft Edge, Safari) จะมีส่วนที่เรียกว่า JavaScript Engine (เช่น engine V8 ของ Chrome หรือ JavaScriptCore ของ Safari) ทำหน้าที่อ่านซอร์สโค้ด JavaScript ที่เราเขียน จากนั้นแปลงเป็นภาษาเครื่อง (Machine Code) ที่คอมพิวเตอร์เข้าใจได้อย่างรวดเร็วผ่านกระบวนการที่เรียกว่า JIT (Just-In-Time) Compilation ทำให้หน้าเว็บสามารถทำงานได้อย่างลื่นไหล

3. โครงสร้างพื้นฐานของ JavaScript

การเชื่อมไฟล์ JavaScript

เราสามารถใส่ JavaScript เข้าไปในโปรเจกต์เว็บได้ 2 วิธีหลักๆ ดังนี้:

1. เขียนแบบภายใน (Internal JavaScript): เขียนภายในแท็ก <script> ในไฟล์ HTML โดยตรง

<!DOCTYPE html>
<html>
<body>
    <h1>ยินดีต้อนรับ</h1>
    <script>
        console.log("Hello from Internal Script!");
    </script>
</body>
</html>

2. เขียนแบบภายนอก (External JavaScript): แยกโค้ด JavaScript ไปไว้ในไฟล์ต่างหาก (นามสกุล .js) แล้วเชื่อมเข้ามาใน HTML วิธีนี้เป็นวิธีมาตรฐานที่แนะนำเพราะช่วยให้โค้ดเป็นระเบียบ

<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
    <script src="script.js" defer></script>


    
ยินดีต้อนรับ

การเขียนคำสั่ง (Statements)

คำสั่งใน JavaScript คือคำสั่งที่บอกให้คอมพิวเตอร์ทำงานบางอย่าง โดยแต่ละคำสั่งมักจะจบด้วยเครื่องหมาย Semicolon ( ; ) (ใน JavaScript ยุคใหม่สามารถละเว้นได้ แต่การใส่ไว้จะช่วยลดข้อผิดพลาดในบางกรณี)

ตัวอย่างการเขียนคำสั่งพื้นฐาน:

  • การประกาศตัวแปร: let name = "John";
  • การแสดงผลบนหน้าจอคอนโซล: console.log("สวัสดีคุณ " + name);
  • การคำนวณ: let total = 10 + 20;

Comment (คำอธิบายโค้ด)

Comment คือส่วนของข้อความที่เราเขียนไว้เพื่ออธิบายโค้ดให้ตัวเองหรือคนในทีมอ่าน โดยที่ JavaScript Engine จะข้ามการประมวลผลในส่วนนี้ไป (โค้ดไม่ทำงาน) มี 2 แบบคือ:

1. Comment แบบบรรทัดเดียว (Single-line Comment): ใช้เครื่องหมาย //

// นี่คือการประกาศตัวแปรเก็บอายุ
let age = 25;

2. Comment แบบหลายบรรทัด (Multi-line Comment): ใช้เครื่องหมาย /* ปิดด้วย */

/*
  ฟังก์ชันนี้สร้างขึ้นเพื่อ
  คำนวณภาษีมูลค่าเพิ่ม (VAT 7%)
  รับค่าเป็นจำนวนเงินสุทธิ
*/
let vat = amount * 0.07;
